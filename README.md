# 🧠 StudyOS — AI-Powered Productivity Assistant

A beautiful, AI-powered productivity assistant built for Diploma Computer Engineering students preparing for **campus placements** and **ECET**.

## ✨ Features

| Feature | Details |
|---|---|
| 🤖 AI Study Plans | Personalized daily plans via Google Gemini |
| ⏱️ Pomodoro Timer | Built-in focus timer with session tracking |
| 📊 Smart Dashboard | Charts, streaks, progress bars |
| 🎯 Goal Tracking | Daily / Weekly / Monthly / Placement goals |
| 📈 Subject Progress | Java, DSA, SQL, Aptitude, English, Project, ECET |
| 🔔 Smart Reminders | Study, sleep, and mobile usage alerts |
| ⚠️ Procrastination Alerts | AI-powered nudges when idle |
| 📅 Auto Timetable | College + self-study time blocked (9:30–4:00 PM) |
| 🗄️ Local Storage | IndexedDB — no server needed |
| 📤 Data Export | Export all your data as JSON |
